AI is pretty smart. But it's also pretty forgetful.

OpenTrawl crawls your messages, mail, calendar, photos and more into plain SQLite archives on your Mac, then puts one search over all of them. Your AI answers from your actual history: one call, no interrogation.

Everything stays on your machine, unless you choose to send it somewhere.

Built for your AI Every command takes --json and returns refs your agent can open. It onboards from your whole history in one pass, not one prompt at a time.

Private by construction Crawling and search run on your Mac. Read commands never write. The archives are plain SQLite files on your disk. Open them yourself.

Quickstart

The toolchain comes from devenv, so you need Nix and devenv installed, plus direnv for the per-terminal setup. Everything builds from source into a repo-local bin directory, so there are no global installs.

git clone https://github.com/opentrawl/opentrawl
cd opentrawl
direnv allow           # or: run `devenv shell` in this directory
scripts/dev-bin        # build every crawler and the trawl CLI into .dev/bin

Then work against your own data. status and search read only; sync is the only command that fetches.

trawl status                 # every source: state, freshness, counts
trawl imessage sync          # crawl one source into its archive
trawl search "boat trip"     # search every source, newest first
trawl open imessage:msg/8842 # expand any ref a search returned
